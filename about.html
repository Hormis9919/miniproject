About the Car Price Predictor

An AI-powered web application for predicting used car prices with high accuracy.

Our model utilizes advanced machine learning techniques such as Random Forest and Extra Trees Regression to analyze historical sales data and estimate resale values.

By inputting details like make, model, year, mileage, and other features, users can get a precise price prediction for a used car.

This project is built using Flask for the backend and Bootstrap for a responsive and user-friendly interface.
